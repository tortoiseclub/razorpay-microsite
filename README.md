# Tortoise × Razorpay — Device Leasing Microsite

Static microsite for Razorpay's Device Leasing Program (powered by Tortoise).

## Pages

- `/` — landing page (`index.html`)
- `/catalog` — device catalogue (`catalog.html`)

## Deploy

Hosted on Vercel. Pushing to `main` triggers production deploys.

## Local preview

Serve the repo root with any static server, e.g. `npx serve .`, then open `/`.
